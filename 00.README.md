# 🌐 Web Development  – HTML, CSS, JavaScript

This repository contains a mini web development basics and  project built using HTML, CSS, and JavaScript. It's designed for practice and learning the core building blocks of modern frontend development.

 🚀 Technologies Used

- 🧱 HTML – Structure of the webpage  
- 🎨 CSS – Styling and layout  
- ⚙️ JavaScript – Interactivity and logic  


 📁 Project Features

- Clean and responsive layout using HTML & CSS
- Basic DOM manipulation with JavaScript
- Includes multiple sections like:
  - Home
  - About
  - Projects
  - Contact Form

🧠 Skills You Can Learn

- Writing semantic HTML
- Styling pages with CSS (inline, internal, or external)
- Adding basic interactivity with JavaScript
- Organizing files in a clean project structure
- Hosting on GitHub Pages
